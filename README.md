## ✨ Features

- **Fully Responsive Design** - Works seamlessly on desktop, tablet, and mobile devices
- **Clean & Professional UI** - Modern design with smooth hover effects and transitions
- **Interactive Elements** - Click animations on projects and internship cards
- **Resume Download** - One-click download of my resume (TXT format)
- **Working Contact Links** - Email, phone, LinkedIn, and GitHub links are functional
- **Split Education Cards** - Distinct visual cards for B.Tech and HSC education
- **Skills Tags** - Visually organized technical skills section
- **Certifications Display** - Clean presentation of professional certifications
- **Dynamic Footer** - Automatically updates with current year
